# Fashion-Recommender-system

A machine learning-based fashion recommendation system that suggests products based on user preferences and image similarity.

## Features

- Fashion product recommendations
- Image-based search system
- Dataset processing and analysis

## Installation

1. Clone the repository
2. Create a virtual environment: `python -m venv .venv`
3. Activate the virtual environment: `.venv\Scripts\activate`
4. Install dependencies: `pip install -r requirements.txt`

## Usage

Run the application:

```bash
python app.py
```

Or use the Jupyter notebooks:

- `main.ipynb` - Main recommendation system
- `fashion_rs.ipynb` - Fashion recommender prototype

## Project Structure

- `app.py` - Main application file
- `fashion_search_system.py` - Search and recommendation logic
- `requirements.txt` - Project dependencies
- `styles.csv` - Product dataset
- `myntradataset/` - Fashion dataset directory
- `images/` - Image resources

## Dataset

The system uses the Myntra fashion dataset for training and recommendations.
